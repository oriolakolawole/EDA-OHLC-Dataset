## Forex Market Exploratory Data Analysis & Feature Engineering (EURUSD)
- Processed multi-timeframe EURUSD OHLC data (15min, 1H, 4H, 1D) by resampling tick data and engineering time-based market features, including session indicators and derived price metrics.
- Performed comprehensive exploratory data analysis (EDA) using statistical summaries, correlation analysis, volatility profiling, and return distributions to uncover trend structures and regime behavior.
- Built reusable data preprocessing pipelines in Python (Pandas, NumPy, Matplotlib, Seaborn) to clean, transform, and visualize time-series financial data for downstream predictive modeling tasks.
